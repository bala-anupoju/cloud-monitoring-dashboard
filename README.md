## 🎯 What It Does

- **Unified view** — Azure, AWS, GCP in one dashboard
- **Real-time alerts** — CPU, memory, disk, network thresholds
- **OpenTelemetry** — distributed tracing across services
- **Cost monitoring** — FinOps alerts for budget overruns
- **SRE metrics** — SLA, uptime, MTTR, MTTD tracking
- **AI anomaly detection** — flag unusual patterns automatically
